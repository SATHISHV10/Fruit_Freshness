# 🍎 Fruit Freshness Detection System

A web-based Fruit Freshness Detection System developed using **HTML, CSS, Python, and Flask**. The application allows users to upload fruit images and determines whether the fruit is **Fresh** or **Rotten**.

---

## 📌 Project Overview

The Fruit Freshness Detection System is designed to help users quickly identify whether a fruit is fresh or rotten. Users upload an image through a simple web interface, and the application compares it with the available dataset to determine the fruit's freshness.

This project demonstrates the integration of **Frontend (HTML & CSS)** with a **Python Flask backend**.

---

## ✨ Features

- 🍎 Attractive and responsive user interface
- 📤 Upload fruit images
- 🟢 Detect Fresh fruits
- 🔴 Detect Rotten fruits
- 📂 Dataset-based image verification
- ⚡ Fast prediction results
- 💻 Simple Flask backend

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/SATHISHV10/Fruit_Freshness.git
```

### Navigate to the project

```bash
cd Fruit_Freshness
```

### Install dependencies

```bash
pip install flask
```

### Run the application

```bash
py app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```

---

## 📷 How It Works

1. Open the application.
2. Upload a fruit image.
3. The application checks whether the uploaded image exists in:

- `dataset/fresh`
- `dataset/rotten`

4. Displays

- 🍏 **FRUIT IS FRESH**
- 🍎 **FRUIT IS ROTTEN**
